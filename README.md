## Управление
### Меню
 - **D-Pad**: Выбор пазла
 - **X**: Начать
 - **O**: Вход во вкладку "Авторы"
### Вкладка "Авторы"
- **O**: Вернуться назад
### Игра
- **D-Pad**: Выбор кусочка
- **X**: Перемещение кусочка
- **O**: Выйти в меню
