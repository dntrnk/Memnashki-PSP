# Memnashki (PSP)
"Пятнашки" с мемами вместо чисел для PSP

![memnashkiScreenshot2](https://github.com/user-attachments/assets/c54e497e-9ef4-453e-b4af-12fd21373396)

## Управление
### Меню
 - **D-Pad**: Выбор пазла
 - **X**: Начать
 - **O**: Вход во вкладку "Авторы"
### Вкладка "Авторы"
- **O**: Вернуться назад
### Игра
- **D-Pad**: Выбор кусочка
- **X**: Перемещение кусочка
- **O**: Выйти в меню
